Debian食用参考教程

第一步 连接网络

  • 1.学校机房
    因为学校机房的某些不可知的原因，DHCP无法使用，所以需要手动配置网络。
    添加一个网络，选择ipv4,方法改为手动，DNS服务器可以采用阿里公用的223.5.5.5，子网掩码为255.255.255.0网关为统一的10.145.85.1（社团机房）地址就是本台电脑的ip（有分配）
  • 2.其他
    除本机房外有线网络或者无线基本可自动连接，如遇困难请参考 wiki.debian.org

第二步 联网校对时间

  • 分别输入以下命令 

     sudo apt install systemd-timesyncd
 systemctl enable systemd-timesyncd
 systemctl start systemd-timesyncd

第三步 换源（默认软件下载源）

  • 由于从官网下载的debian默认下载源为国外，可能会导致无法下载软件所以需要换成国内高校的镜像源。附上链接 换源 好用的复制扩展 allow copy

第四步 安装并学习基本命令

写在前面：
1.debian命令最好加上sudo
2.打开终端快捷键Ctrl+Alt+T (打开后默认位于用户目录)

正文

  • apt
    例如：sudo apt-get install gcc (debian最常用的安装软件方法用于安装软件库里的软件)
  • wget 需要安装（sudo apt-get install wget)
  • dpkg 第二常用的包安装命令，用于安装后缀名为 .deb 的文件（需要在目标目录打开终端）
    例如：sudo dpkg -i [软件包名]
    tips：输入软件包名可以使用tab键，输入首几位字母后用tab补全

记得加sudo！！！

第五步 安装一些重要的软件

给出一个参考列表：
1.vscode（需要去下载deb包）
2.git（不多解释）
3.icalingua++ (qq的linux替代版)  
4.gparted (一个强大的分区软件，可能会用到)
5.g++ (c++的编译器可以编译c)
6.gdb (调试器)
