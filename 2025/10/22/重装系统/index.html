介于第一步和第二步中间的一步 备份数据

来源：联想知识库

一、需备份的内容清单📁

1. 个人文档与文件

  • Word、Excel、PPT 等办公文档
  • 编程代码、设计稿、研究资料
  • 合同、证书、报告等重要资料

2. 图片与视频

  • 家庭照片、旅游视频等珍贵回忆
  • 建议集中存放并多点备份（如移动硬盘、云盘）

3. 下载文件

  • 浏览器、微信、QQ 等下载的文件，默认在“下载”文件夹中

4. 桌面文件

  • 桌面文件默认存储在系统盘，重装后易丢失，需提前转移

5. 软件与应用程序设置

  • AppData、ProgramData 文件夹中保存的软件配置，如Vscode配置文件等
  • Photoshop、IDE 等程序的个性化设置

二、如何找到它们并备份

1、C盘内文件

C:\Windows

  • 操作系统核心目录，包含系统运行所需的文件。
  • 子目录：
    • System32：系统核心组件和 DLL 库。
    • SysWOW64：32 位程序在 64 位系统上的支持库。
    • Fonts：系统字体。
    • Temp：临时文件。

C:\Program Files

  • 默认安装 64 位程序 的位置。

C:\Program Files (x86)

  • 默认安装 32 位程序 的位置。

C:\ProgramData

  • 所有用户共享的程序数据和配置文件（默认隐藏）。

C:\Users

  • 存储所有用户的个人数据。
  • 子目录：
    • 用户名\Desktop：桌面文件。
    • 用户名\Documents：文档。
    • 用户名\Downloads：下载内容。
    • 用户名\AppData：用户级程序配置（含 Local、Roaming、Temp）。

2、除C盘外的文件

  • 若C盘分配大小足够，只需格式化C盘进行重装系统，则无需备份除C盘外文件
  • 若需重新划分C盘，则应按需备份其他盘内文件

非常重要的一点！！！在安装开始之前，请确定硬盘无Bitlocker锁，以免造成不必要的损失 检查与关闭的方法

  • 以截图为主





  • 选择要安装的系统的系统盘，请仔细辨认安装位置盘符，以免装在启动盘上或破坏其他盘的数据。
  • 如果不保留其他盘或需要依靠重装扩大C盘分区，建议将之前的分区全部删除，避免系统重装过程中产生bug（记得备份好数据）。

  • 可以看到，“未分配的空间”字样已经没有了，说明分区成功。

✅小知识

1、分区是建立在物理硬盘基础上的，单个分区的大小受物理硬盘的限制，无法将两块硬盘划为一个分区。

  • 老款台式固态硬盘容量较少或只有一块较小容量的硬盘，所以一般将系统分区建立在固态硬盘上以提高速度（文件等存储在机械硬盘也就是另一个分区上，），但这也导致了若使用不当C盘易爆满。对于比较新的机子，固态基本都有1t容量，不进行分区也可使用，不过为了整理文件，选择分区再开一个盘符。

2、磁盘类型解释：

  • 系统分区：ESP分区 负责存放EFI引导文件
  • 主分区：Windows11系统主分区 系统就放在这里面
  • 恢复：RE恢复分区 负责存放系统恢复文件

3、常见文件系统格式对比

文件系统最大单文件最大分区兼容性安全功能适用场景
FAT324GB2TB✅ 高❌ 无权限U盘、小文件传输
NTFS~16TB~256TB⚠️ Windows优先✅ 加密/权限/压缩系统盘、大文件
exFAT理论无限理论无限✅ 高⚠️ 无权限移动硬盘、跨平台传输

简要说明

  • FAT32：兼容性好，但不支持大文件。
  • NTFS：功能强，适合 Windows 系统盘。
  • exFAT：支持大文件，适合移动设备和跨平台。

建议：U盘用 exFAT，系统盘用 NTFS，老设备用 FAT32。

第四步 安装驱动

核心原则：先打通网络 → 再装主板芯片组 → 再装显卡/声卡/其它 → 最后用设备管理器查漏补缺 每安装完一个重启一次最好
若以稳定性优先，可以先上官网查询对应型号的各驱动版本再进行下载，或者直接下载对应品牌的电脑管家。
来源：CSDN

第一步：连接网络（一切的前提）

1. 重启电脑，检查是否已自动装好基础网卡驱动

  • 有线：插网线，看能否自动获取 IP 并上网
  • 无线：观察任务栏是否有 Wi-Fi 图标，能否扫到热点

2. 若网络不通（最常见）

① 用另一台设备访问主板官网（手机/笔记本/平板均可）
- 华硕 https://www.asus.com.cn/support
- 微星 https://www.msi.cn/support
- 技嘉 https://www.gigabyte.cn/support
- 华擎 https://www.asrock.com.cn/support/index.cn.asp
② 查找主板** exact 型号（盒子、PCB、购买记录都能找到）
③ 在对应产品页 → 驱动下载 → 选 OS 版本 → 下载
- 有线：LAN Driver
- 无线：Wireless LAN Driver / Wi-Fi Driver
④ U 盘拷回新电脑 → 运行安装包（zip 就先解压再 Setup.exe）
⑤ 装完务必重启** → 验证网络是否恢复

3. 临时救急法

  • 手机数据线 + USB 网络共享，也能让 PC 先上网

第二步：主板芯片组（Chipset/INF）

  1. 回到主板官网 → 下载 Chipset Driver / INF Driver（体积通常最大）
  2. 管理员身份运行 → 一路下一步 → 重启
  3. AMD Ryzen 用户可选：直接到 AMD 官网 下最新版，往往比主板页更新

第三步：显卡驱动

显卡品牌下载地址备注
NVIDIAhttps://www.nvidia.cn/Download/index.aspx?lang=cnGame Ready / Studio 二选一，建议勾“清洁安装”
AMDhttps://www.amd.com/zh-hans/supportAdrenalin 版
Intel 核显 / Archttps://www.intel.cn/content/www/cn/zh/download-center/products.html选“显卡”分类

安装完 → 重启

第四步：声卡驱动

  • 主板官网 → Audio Driver（多为 Realtek）
  • 运行 Setup → 重启

第五步：其它可选但推荐

  • 蓝牙：Bluetooth Driver
  • Wi-Fi：若第一步只装了有线，这里补无线
  • Intel ME/AMD PSP：主板页的 Management Engine / PSP 驱动
  • SATA/RAID：现代系统一般自带，遇硬盘异常再装
  • RGB 灯控：Armoury Crate、Mystic Light、RGB Fusion 等（装完软件即自动打驱动）

第六步：设备管理器查漏补缺

  1. Win + X → 设备管理器
  2. 看是否有黄色感叹号/问号
    • 右键 → 更新驱动 → 自动搜索（需网络）
    • 自动失败就手动：
      ① 确认硬件 ID / 名称
      ② 回主板官网找对应驱动
      ③ 手动指定路径安装

安装完驱动后千万别忘记重启！

第五步 激活系统

  • 市面上台式与笔记本一般自带绑定的激活码，安装后无需手动激活。这里分享无激活码的情况。
  • 采用HEU KMS Activator进行激活，点击下载

  • 首先选择智能激活，点击右下角图标查看是否激活，若未激活，右侧框内勾选激活windows，数字/KMS38即可

Tips:

1、安装完成后最好关闭windows更新

2、非常重要的一点！！！在安装开始之前，请确定硬盘无Bitlocker锁，以免造成不必要的损失 检查与关闭的方法

3、如果无法进入Bios或者快速启动选项，请检查显示器是否连接正确（若是无核显的机子，显示信号传输线应该连在显卡上 **这是一个惨痛的教训**）
